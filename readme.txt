Este programa es un proyecto para entender la construcción de procesadores de lenguajes de 
programación de alto nivel, mediante la creación de un compilador simple. El nombre del 
proyecto es "Desarrollando un Mini-Compilador de Expresiones Algebraicas" y es el tercer 
proyecto del curso de Compiladores e Intérpretes del Instituto Tecnológico de Costa Rica. En 
este caso la tarea se ha realizado en el lenguaje "Java".

Objetivo General:
Aplicar principios, modelos y técnicas para el diseño y la construcción de procesadores de
lenguajes de programación de alto nivel, con énfasis en compiladores e intérpretes, para este
proyecto, enfocándose en el diseño de un compilador simple.

Para ejecutar esta aplicación se debe realizar lo siguiente:

Compilar el programa de la siguiente manera: 
javac MiniCompilador.java

Ejecutar el programa de esta manera:
java MiniCompilador entrada.txt

Siendo entrada.txt el archivo con un código con la gramática especificada en el documento del 
proyecto. Cabe destacar, que los archivos con los que se trabajará deberán estar en el mismo 
directorio en el que se encuentra la clase principal "MiniCompilador.java".
